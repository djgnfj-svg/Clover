## 💎기능
### 유저
- JWT + Refresh token 로그인
- 회원가입
- 마이페이지
### 클럽
- Club Search
- Club 가입/탈퇴
- Club 관리자(Manager, Master) 기능
    * Master
        - 공지사항 CRUD (추후 업데이트)
        - 클럽 프로필 수정 (이미지 , 주제 , 소개)
        - 클럽 소개 수정
        - 가입관리 ( 가입신청한 유저 가입 및 거절)
        - 유저관리 (유저 제명 , 매니저 승급 & 마스터 인계 , 유저 정보 )
        - 클럽 해체하기 
    * Manager
        - 공지사항 CRUD ( 추후 업데이트 )
        - 가입관리
